Introduction

jQuery is a quick, small and rich quality JavaScript library. jQuery helps to make things easier for event handling, HTML document traversal and manipulation, animation and Ajax, a lot smooth with effortless API that works beyond a multitude of browsers. JQuery has switched the way of writing JavaScript with the combination of adaptability and extensibility.

Advantages

  • Time Saving: Five lines of jQuery are identical to 25 lines of ordinary JavaScript code. This implies loading web pages quicker and storing files in small sizes.
  • Huge Community: There is a huge community help network on the web to help you rapidly cure any bug issues. There are tons of plug-ins to help you abundantly.
  • Search Engine Optimization: While search engines are showing signs of improvement at having the option to read the content within some Flash, the text is already set up inside the jQuery. This aids that it is totally clear to all the search engines, uncovering all your keyword-rich content.
  • Plugin Support: There are plenty of plug-ins on the web that can create basic and quick extraordinary effects for the web developers.
  • Browser and Mobile Friendly: jQuery is one of the most popular JavaScript libraries and mostly works in all browsers. Even it is highly supported by any mobile device where the web browser is compatible with JavaScript.
  • Open Source: Basically it is an open-source software free of cost. That means it can be modified and can be managed according to the demands of the developer.

Disadvantages

  • Increased technology stack: At the point when you add a structure or toolbox to any project, you should always pause and think in the event that it is worth it or not. You will include dependency and prospect developers who would need to look after it. jQuery will be around for quite a while, however until the end of time? I don't think so.
  • Bandwidth:It takes up a limited amount of bandwidth to download for the user of your website.
  • Slow down: It additionally (when utilized) slows down the operation of certain tasks - one could/can contend that a few tasks are done quick and reliable in jQuery related with a native plain javascript execution.
  • Better alternatives: There are numerous different frameworks out there, that don't require jQuery and in case you're as of now utilizing one of these, including jQuery may be an anti-pattern or simply including additional complexity that wasn't required.

Usages

  • JQuery can be utilized to create Ajax based applications.
  • It can be utilized very well to make code basic, concise, and reusable.
  • It simplifies the procedure of traversal of the HTML DOM tree.
  • It can likewise deal with handling events, animation, and include ajax support in web applications.

Program 1:Background Change

This code example uses the click action of jQuery to change the background color of the div (with id example1Container). In the code, three random numbers are created in the range 1-255 to represent red, green and blue colors and then these numbers are used to in the click action to change the background color to a random background. For this particular program, rgb (red, green, blue) function from css has been used.

Program 2: Append Nodes

This program uses append action to add h5 tags in the div with id 'example4Container'. When the Add button with id 'appendbtn' is clicked we append a node h5 tag with class 'ex4' to the parent div (id is example2Container). The Remove Element button with id 'removebtn' uses the remove action to remove the elements added by the add element button. When the Remove element button is clicked it removes the first element of class 'ex4' (selector is ".ex4":nth(0)).
